# Product Overview

iCare is a comprehensive community management platform designed for Malaysian political constituencies (ADUN/DUN areas). The system serves multiple user types:

## Core Features

- **Community Issue Reporting**: Citizens can report local issues with location tracking and media uploads
- **Household Management**: Track household demographics, income, and aid distribution
- **SPR Voter Management**: Manage electoral roll data with support status tracking
- **Aid Programs**: Coordinate and track distribution of government assistance programs
- **Membership Management**: Handle political party membership applications and records
- **Multi-role Access Control**: Different interfaces for administrators, staff, and community members
- **Audit Logging**: Comprehensive activity tracking for accountability and compliance
- **Dynamic Page Builder**: Create and manage public-facing content pages
- **Reporting & Analytics**: Generate insights on demographics, aid distribution, and issue resolution
- **Data Import/Export**: CSV import/export functionality for bulk operations
- **Backup System**: Database backup and restore capabilities

## User Roles

- **Super Admin**: Full system access and configuration
- **ADUN**: Elected representative with oversight capabilities  
- **Zone Leaders**: Manage specific geographical zones
- **Staff**: Handle day-to-day operations and data entry
- **Community Members**: Report issues, apply for aid, view announcements

## Geographic Structure

The system is organized hierarchically: Parliament → DUN → Zone → Village/Cawangan → Household
